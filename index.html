CAMERAS API DOCUMENTATION

Routes

All HTTP methods are supported. You can use http or https for your requests.

Get product (s)

  • Get 10 products by page number:/cameras?page=number (GET)
  • Get X products in each page:/cameras?perpage=number (GET)
  • Search products by name or info:/cameras?s=key-word (GET)
  • Sort products:/cameras?sort=sort-key (GET)
  • Reverse by sort key:/cameras?r=yes (GET)
  • Price range:/cameras?min=number&max=number (GET)
  • Get all products by category:/cameras/category/:category-name (GET)
